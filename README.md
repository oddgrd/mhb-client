# Myhomeboard Climbing PWA Client

## Develop locally:

1. Install pnpm:
   ```bash
   npm install -g pnpm
   ```

2. Install dependencies:
    ```bash
    pnpm i
    ```

3. Start development server:
    ```bash
    pnpm dev
    ```

## Build:

```bash
pnpm build
```

## Test:

With interactive browser:
```bash
pnpm cypress
```

Headlessly:
```bash
pnpm cypress:headless
```
